Типовая задача со структурой данных «приоритетная очередь» на ЯП Python

Для реализации приоритетной очереди в Python применяется модуль heapq, который предоставляет функции для работы с очередью на основе бинарной кучи. По умолчанию создается min-heap, где наименьший элемент имеет высший приоритет. Для изменения порядка сортировки можно использовать пользовательские классы с переопределенными методами сравнения.
